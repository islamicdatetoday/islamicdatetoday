# IslamicDate.Today

IslamicDate.Today is a reference platform for Islamic calendar systems and Hijri date data.

This project publishes datasets, documentation, and reference tools related to Islamic dates and the Hijri calendar. It is designed to be a long-term, stable, and neutral technical reference for developers, researchers, and users.

Official website:
https://islamicdate.today

---

## Purpose

The purpose of this project is to:

- Provide machine-readable Hijri date datasets
- Document Islamic calendar and Hijri date concepts
- Describe how dates are determined and maintained
- Provide stable definitions and reference documentation
- Publish reference tools that use the data
- Serve as the technical and data backend for https://islamicdate.today

This repository is not a blog and not a news site. It is a reference, data, and documentation project.

---

## Repository Structure

### Data (/data/)
Contains machine-readable datasets published by IslamicDate.Today, including:
- Hijri date tables
- Metadata files
- Dataset descriptions

---

### Definitions (/definitions/)
Contains reference-style documents that explain:
- What an Islamic date is
- What the Hijri calendar is
- What an Islamic calendar system is

---

### Methodology (/methodology/)
Contains documents that explain:
- How dates are determined
- How data is produced and updated
- What sources and methods are used

---

### Documentation Site (/docs/)
Contains the source files for the public documentation website (GitHub Pages).

---

### Tools and Code (/src/)
Contains reference implementations and tools, including:
- A date conversion tool
- Example code for loading and using the dataset
- Demonstrations of how the data can be used in applications
