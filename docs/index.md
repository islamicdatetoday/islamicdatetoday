# IslamicDate.Today Documentation

Welcome to the official documentation site for the IslamicDate.Today project.

This documentation provides:

- Definitions of Islamic date and Hijri calendar concepts
- Description of the data published by IslamicDate.Today
- Methodology notes explaining how dates are determined
- Reference information and project scope

## Project Purpose

IslamicDate.Today is a reference platform for Islamic calendar systems and Hijri date data.

The project publishes:

- Machine-readable datasets
- Stable definitions
- Methodology documentation
- Reference material for developers, researchers, and users
