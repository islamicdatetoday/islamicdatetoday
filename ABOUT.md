# About IslamicDate.Today

IslamicDate.Today is a reference and data platform focused on Islamic calendar systems and Hijri date information.

Its purpose is to document, standardize, and publish information about how Islamic dates are defined, calculated, and used across different systems and regions.

The platform covers, among others:

- The Umm Al Qura calendar used in Saudi Arabia
- Observational Hijri calendars based on moon sighting
- Astronomical Hijri calendars based on calculations
- Date conversion methodologies between Gregorian and Hijri calendars

IslamicDate.Today aims to provide:

- Stable definitions
- Verifiable datasets
- Transparent methodology documentation
- Neutral, non-sectarian, non-political reference material

This repository serves as the official documentation and data source for the IslamicDate.Today website.
